Universal RNG

Universal RNG is a high-performance, high-quality Random Number Generation (RNG) library optimized for Unity game development. Powered by Rust

Providing performance unattainable with standard UnityEngine.Random or System.Random, Universal RNG proves its true value in high-density simulations and complex procedural content generation.

Installation

Install via Unity Package Manager (UPM) or import the provided .unitypackage.

Basics

All RNG classes inherit from IDisposable. Using the using statement is recommended to ensure native memory is properly released. 

using Cet.Rng;

// Mersenne Twister (32-bit): Initialize with seeds
using var mt = new Mt19937(1, 1024);
Debug.Assert(mt.Next() == 244660247);
// Batch generation (Returns Span<uint>)
Debug.Assert(mt.NextU32s(3).SequenceEqual(new uint[] 
{ 
    1183135446, 
    982747242, 
    221797415 
}));

// Sfc64 (64-bit): High-speed, chaotic PRNG
using var sfc = new Sfc64(1);
Debug.Assert(sfc.Next() == 4613303445314885483);
Debug.Assert(sfc.NextU64s(3).SequenceEqual(new ulong[] 
{ 
    9250377774926580674, 
    11474594222839663884, 
    10182195164632316717 
}));

// PCG (32-bit): Statistically excellent PRNG
using var pcg = new Pcg32(1);
Debug.Assert(pcg.Next() == 4164751464);
Debug.Assert(pcg.NextU32s(3).SequenceEqual(new uint[] 
{ 
    4202279039, 
    2817093042, 
    1081118559 
}));

Batch Generation

The standout feature of Universal RNG is its ability to generate multiple random numbers in a single call. This minimizes managed-to-native call overhead, resulting in extreme performance. Results are returned as ReadOnlySpan<T> for efficient, zero-copy access. 

// Generate 1000 random numbers at once (minimizes Managed-to-Native overhead)
ReadOnlySpan<float> batch = rng.RandF32s(1000, 0f, 1f);

foreach (var value in batch)
{
    // Efficient access without copying
}

Available Algorithms

Choose the best algorithm for your specific use case.

Philox32 / Philox64

Counter-based RNG. Ideal for parallel processing, providing excellent reproducibility in GPU and multi-threaded environments.

Xorshift32 / Xorshift64 / Xorshift128

Ultra-lightweight algorithms. Extremely low memory footprint and blazing fast performance.

Mt19937 / Mt1993764

Traditional Mersenne Twister. Very long periods and excellent statistical properties.

Pcg32

A modern algorithm with a perfect balance of unpredictability and speed.

Sfc64

"Small Fast Chaotic" PRNG. Extremely fast and memory-efficient, ideal for simulations in 64-bit environments.

Cet64

A custom 64-bit generator combining advanced bit rotation and multiplication. Delivers high randomness through complex mixing.

IRng32 Reference

ReadOnlySpan<uint> NextU32s(int count)

Generates the specified number of 32-bit unsigned integers.

ReadOnlySpan<float> NextF32s(int count)

Generates the specified number of floats between 0.0 and 1.0.

ReadOnlySpan<int> Randints(int count, int min, int max)

Generates a batch of integers within the range [min, max].

IRng64 Reference

ReadOnlySpan<ulong> NextU64s(int count)

Generates the specified number of 64-bit unsigned integers.

ReadOnlySpan<double> NextF64s(int count)

Generates the specified number of doubles between 0.0 and 1.0.

ReadOnlySpan<long> Randints(int count, long min, long max)

Generates a batch of 64-bit integers within the range [min, max].
