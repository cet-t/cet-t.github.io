Seamless Integration

Intuitive API

Simple, thread-safe API design allows for integration into existing projects in minutes. 

// Simple Initialization
using var rng = new Philox32(seed);

// Get Single Value (0.0 ~ 1.0)
float val = rng.Value();

// Range Selection [min, max] / [min, max)
float f = rng.Randfloat(0f, 100f);
int i = rng.Randint(0, 10);

// High-speed Batch Generation (via ReadOnlySpan)
var batch = rng.Randfloats(1000, 0f, 1f);
