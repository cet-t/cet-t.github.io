Unilox

Uniloxは、Unityでのゲーム開発に最適化された超高速乱数生成（RNG）アセットです。
核心部はRustで実装されており、メモリ安全性とネイティブ級の実行速度を両立しています。

強力なバッチ生成機能を備え、数百万の個体に対する乱数生成を瞬時に完了させます。

圧倒的な実行速度

Uniloxは、標準の System.Random や UnityEngine.Random を凌駕するパフォーマンスを提供します。 内部に Xorshift や Philox アルゴリズムを搭載し、SIMD最適化が施されています。

Unilox (Philox32) 62.4943 ms
UnityEngine.Random 723.6959 ms
System.Random 885.5966 ms

※ 1億個の生成を10回試行した際の平均時間。ネイティブ環境での比較。

Seamless Integration

直感的な使用例

シンプルかつスレッドセーフなAPI設計により、既存のプロジェクトに数分で組み込むことが可能です。 

// 非常にシンプルな初期化
using var rng = new Philox32(seed);

// 単一値の取得 (0.0 ~ 1.0)
float val = rng.Value();

// 範囲指定の取得 [min, max] / [min, max)
float f = rng.Randfloat(0f, 100f);
int i = rng.Randint(0, 10);

// 強力な超高速バッチ生成 (ReadOnlySpanでの取得)
var batch = rng.Randfloats(1000, 0f, 1f);
