Uniloxは、Unityでのゲーム開発に最適化された超高速乱数生成（RNG）ライブラリです。
核心部はRustで実装されており、メモリ安全性とネイティブ級の実行速度を両立しています。

実行速度の比較

Uniloxは、標準の UnityEngine.Random や System.Random を凌駕するパフォーマンスを提供します。 内部に Xorshift や Philox アルゴリズムを搭載し、SIMD最適化が施されています。

Unilox (Philox32)62.5 ms
UnityEngine.Random723.7 ms
System.Random885.6 ms

※ 1億個の生成を10回試行した際の平均時間。

直感的な使用例

シンプルかつスレッドセーフなAPI設計により、既存のプロジェクトに数分で組み込むことが可能です。 

// 非常にシンプルな初期化
using var rng = new Philox32(seed);

// 単一値の取得 (0.0 ~ 1.0)
float val = rng.Value();

// 範囲指定の取得 [min, max]
int i = rng.Randint(0, 10);

// 強力な超高速バッチ生成 (ReadOnlySpanでの取得)
var batch = rng.Randfloats(1000, 0f, 1f);
